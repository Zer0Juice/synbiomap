# Beamer style guide

## Slide types
- title
- roadmap
- setup
- mechanism
- identification
- result
- robustness
- takeaway
- conclusion

## Recommended section flows

### Conference talk
1. Question
2. Mechanism
3. Data/design
4. Main result
5. One robustness block
6. Conclusion

### Job talk
1. Big question
2. Contribution
3. Mechanism
4. Data/design
5. Main results
6. Heterogeneity/extensions
7. Broader agenda

## Density rules
- no paragraph blocks
- no 8-row regression tables unless rebuilt for slides
- no more than one major equation per slide
